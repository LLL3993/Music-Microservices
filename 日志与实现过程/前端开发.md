# 前端开发（music-web）

### 目标
在 `music-web` 目录下使用 `Vue3 + Vite + Vue-Router@4 + Axios` 搭建一套「QQ 音乐」风格的静态站点（浅色为主，可切换深色模式），包含 4 个路由页面：
- `Layout`：顶部栏 + 左侧导航 + 右侧内容区 + 底部播放器占位
- `Search`：热门标签 + 搜索输入 + 结果列表（假数据）
- `Player`：全屏弹层播放页（假数据），支持封面旋转与歌词高亮
- `Empty`：404 兜底

### 项目结构（关键目录）
`music-web/src`：
- `main.js`：应用入口、主题初始化、挂载路由
- `router/index.js`：路由配置（Layout 嵌套路由 + Player + 404）
- `views/Layout.vue`：应用骨架（Topbar/Sidebar/Content/Playerbar）
- `views/Search.vue`：搜索页（热门标签、回车跳转、假数据列表）
- `views/Player.vue`：播放页（全屏遮罩、封面旋转、歌词 5 行、控制区占位）
- `views/Empty.vue`：404 页面
- `style.css`：全局主题变量与基础样式

### 路由设计
- `/` → `Layout`（容器）
  - `/search`（name: `Search`）→ `Search.vue`
  - `/` 默认重定向到 `/search`
- `/player`（name: `Player`）→ `Player.vue`
- `/:pathMatch(.*)*`（name: `Empty`）→ `Empty.vue`

### 明暗主题切换
- 主题标识：写入 `document.documentElement` 的 `data-theme="light" | "dark"`
- 持久化：`localStorage.setItem('theme', theme)`
- 默认策略：
  - 优先读取 `localStorage.theme`
  - 否则根据 `prefers-color-scheme` 选择 `light/dark`
  - 最终回退到 `dark`
- 主题变量：`music-web/src/style.css`
  - 深色主：`#121212`
  - 浅色主：`#f5f5f5`
  - 强调色：`#ff4e4e`
  - 圆角：`12px`
  - 阴影：`0 2px 8px rgba(0,0,0,.06)`

### 关键交互（均为占位逻辑）
- 搜索回车：
  - 触发跳转：`/search?q=${encodeURIComponent(keyword)}`
  - 搜索结果：本地假数据列表根据关键字简单过滤
- 播放按钮：
  - `@click="goPlayer(name)"`，跳转 `/player?name=...`
- 播放页封面旋转：
  - 播放状态为 `true` 时添加类 `playing`
  - 动画：`animation: rotate 20s linear infinite;`
- 歌词滚动/高亮：
  - 假数据 5 行
  - 通过 `setActiveLine(index)` 切换 `active` 行（点击行触发）
- 关闭播放页：
  - 右上角关闭按钮：优先 `router.back()`，无历史则回到 `Search`

### 构建与验证
在 `music-web` 目录执行：
- 开发：`npm run dev`
- 构建：`npm run build`
- 预览：`npm run preview`
